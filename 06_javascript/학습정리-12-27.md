### 1. Bom 이란?
```
Browser Object Model,
	브라우져와 관련된 객체를 말함.
	객체 종류 : 
		window 객체
		screen 객체
		location 객체
		history 객체
		navigator 객체
```
### 2. Dom 이란?
```
Document Ojbject Model ,
태그를 메모리에 올린 것 또는 메모리에 올라간 태그들.

2진 트리 형태로 메모리에 올라간다.

javascript를 통해서 html 태그 객체를 생성, 추가, 삭제, 이동 작업을 할 수 있다.

( ->  동적으로 <ol> <li> 를 추가 할 수 있도록 DOM 객체에서 지원을 한다. )
```
