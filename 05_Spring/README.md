	Authentication :
		인증 과정,

		권한 부여,

			Netflix : 유료 구독자만 영화와 프로그램을 볼수 잇도록 허용.
		승인은 대부분의 웹 사이트에서 핵심기능임 ( 대부분의 웹 사이트는 서비스를 특정 사용자 그룹으로 제한을 함)
