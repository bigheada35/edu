### --클래스 작성 순서--
```
1.클래스 명을 만든다.
2.데이터 멤버(인스턴스 변수)를 만든다.
3.생성자 만든다.
4.데이터 멤버를 컨트롤 하는 함수를 만든다.
5.다른 소스 클래스를 그냥 대놓고 빼겨 본다.
```

### --컴파일러가 공짜로 해 주는것--
```
1.자동 형변환
2.디폴트 생성자
3.상속시 디폴트 super생성자
4.extends Object
```
