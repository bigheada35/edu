### 주의 :
	&& 연산에서는
		앞의 것이 false이면 뒤의 것을 진행하지 않는다.
	
	|| 연산에서는
		앞의 것이 true이면 뒤의 것을 진행하지 않는다.
		
### 주의 : 
```
	short num2 = 7;
	num2 = +num;
```
		+num 에서  +가 num앞에 붙으면
		short가 int로 자동형변환이 된다.
		그러나 num2가 short 이기 때문에 에러가 발생한다.
		
### 주의 :
	-8 >>> 3  : 쉬프트시 MSB를 0 으로 채움
	
	-8 >> 3  : 쉬프트시 MSB를 부호비트와 같은 값으로  채움
				
	-8 <<< 3  : 쉬프트시 LSB를 0 으로 채움
	
	-8 << 3  : 쉬프트시 LSB를 0 으로 채움
