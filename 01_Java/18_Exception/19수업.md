	영역 :		저장내용: 
	------------------
	Method 영역: 	메소드의 바이트코드(*), static변수(****)
	
	Stack 영역: 	지역변수, 매개변수
	
	Heap 영역: 		인스턴스가
				(가비지 컬렉션의 대상)
