#### 작성:
```java
public class Test5 {

	public static void main(String[] args) {
		TreeMap<Integer, String> map = new TreeMap<>();
		map.put(45, "Brown");
		map.put(37, "James");
		map.put(23, "Martin");
		
		Set<Integer> set = map.keySet();
		for( Iterator<Integer> it = set.iterator(); it.hasNext(); ) {
			Integer i = it.next();
			System.out.println(i + ", " + map.get(i));
		}
		
	}
}
```
